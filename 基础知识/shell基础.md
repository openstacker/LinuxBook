## 环境变量及shell相关

### export
主要用于设置和查看环境变量
```
# 设置 PATH  环境变量
$export PATH=$PATH:/opt/stack
# 打印出设置的环境变量
$export 
```
### printenv
打印环境变量
```
# 打印 HOME 环境变量
$printenv HOME

# 打印所有环境变量
$printenv
```
### declare/set

declare 用来声明shell中的变量。

set 用来设置 shell 中选项。
